# 🛍️ Online Store - Angular Shop

A complete e-commerce project built with Angular 20

![Angular](https://img.shields.io/badge/Angular-20.3.0-red)
![TypeScript](https://img.shields.io/badge/TypeScript-5.9.2-blue)
![License](https://img.shields.io/badge/License-MIT-green)

## 🚀 Installation and Setup

```bash
git clone https://github.com/nahidmoradi/shop-angular.git
cd shop-angular
npm install
npm start
```

## ✨ Features

- ✅ Standalone Components Architecture
- ✅ Lazy Loading Routes
- ✅ JWT Authentication
- ✅ Admin Panel with full CRUD
- ✅ Route Guards (Auth, Admin, Guest)
- ✅ HTTP Interceptor
- ✅ Responsive Design
